# Karmada (karmada)
Karmada is a CNCF incubating multi-cluster Kubernetes management system. It provides a unified control plane for scheduling, failover, and traffic management across multiple clusters using Kubernetes-native APIs.

## APIs

### Karmada Multi-Cluster API
Karmada extends Kubernetes with CRDs for multi-cluster management including PropagationPolicy, OverridePolicy, and ResourceBinding for distributing and managing workloads across clusters.
